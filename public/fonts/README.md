# Fonts

## Clash Display Font

This portfolio uses the Clash Display font for headings. 

### Installation Options:

**Option 1: Download the font**
1. Download Clash Display from: https://www.fontshare.com/fonts/clash-display
2. Place `ClashDisplay-Variable.woff2` in this directory

**Option 2: Use fallback**
The site will use system fonts as fallback if Clash Display is not available.

### Current Setup:
- The layout.js file is configured to load Clash Display
- If the font file is missing, it will gracefully fall back to sans-serif
